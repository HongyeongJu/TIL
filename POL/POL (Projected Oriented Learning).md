# POL (Projected Oriented Learning)

- 프로젝트를 만들면서 프로그래밍을 배우자.

## 주의해야하는 원칙들

1. 매일 매일 꾸준히가 중요하다.
2. 처음부터 큰 프로젝트를 만든다고 생각하지말자. (쉬운것부터)
3. 기발한 아이디어를 생각나면 프로젝트 리스트에 적어본다
4. 무조건 기한내에 끝내야된다.
